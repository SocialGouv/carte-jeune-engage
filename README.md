## Developpement

### Webapp

Accédez au dossier de l'application NextJS webapp :

```bash
cd webapp
```

Copiez le fichier .env.example :

```bash
cp .env.example .env
```

Installez les dépendances nécessaires :

```bash
yarn
```

Lancez l'application, qui sera accessible sur le port 3000 :

```bash
yarn dev
```
